Documentación CosmeticLAB

Esta página resume de forma clara el contenido del proyecto para presentarlo de manera ordenada desde la presentación principal.

1. Objetivo del proyecto

Desarrollar una plataforma web para la gestión de productos cosméticos, con enfoque en catálogo, pedidos y administración.

2. Requisitos principales

  • Catálogo interactivo y organizado.
  • Gestión de usuarios y productos.
  • Diseño responsive y accesible.
  • Aplicación escalable y fácil de mantener.

3. Tecnologías utilizadas

  • HTML5, CSS3 y JavaScript.
  • PHP 8 y MySQL.
  • Bootstrap y XAMPP.
